### Инструкция для работы с текстом
***
* Для выделения курсивом, ставим * или _(*Например* или _так_)
* Чтобы выделить текст полужирным, ставим ** или __(**Например** или __Так__)
* Чтобы выделить полужирным курсивом, ставим *** (***Например***)

# Работа с удалёнными репозиториями
  Для работы и предложения изменений файлу другого пользователя необходимо:
    1. Создать (fork) от его репозитория и скопировать ссылку на него в своём профиле
    2. Зайдя в терминал прописываем следующие команды:
       * git clone + ссылка
       * cd [имя папки] чтобы появилась надпись (main)
    3. Создаём новую ветку, в которой и будем производить все необходимые изменения
    4. Выполнить необходиму работу
       * git add .
       * git commit -m "Комментарий"
    5. Для предложения изменений
       * git remote
       * git push origine "Имя ветки"
    NB! В случае ошибки git сам подскажет какую команду необходимо ввести для предложения изменений
    6. Заходим на GitHub, обновляем и, если все команды выполнены правильно, у нас появится кнопка "Compare and pull request"
